# Task 1
Iris Flower Classification using Machine Learning<br>
Overview<br><br>

This project focuses on classifying Iris flowers into three species: Setosa, Versicolor, and Virginica using Machine Learning techniques. The classification is based on four flower measurements:
<br>
Sepal Length<br>
Sepal Width<br>
Petal Length<br>
Petal Width<br>
<br><br>
The objective is to train a machine learning model that can accurately predict the species of an Iris flower based on these measurements.
